# A股个股Beta系数估计与投资组合分析

## 项目概述

本项目基于CAPM模型，对选定的5只A股个股进行Beta系数估计，并构建投资组合进行风险收益特征分析。

## 股票选择

我们选择了覆盖不同行业的5只A股股票：

| 股票代码 | 股票名称 | 所属行业 | 行业特征 |
|---------|---------|----------|----------|
| 000001 | 平安银行 | 银行 | 周期性，资本密集型 |
| 000858 | 五粮液 | 消费 | 消费防御性 |
| 000002 | 万科A | 地产 | 周期性 |
| 300059 | 东方财富 | 金融科技 | 成长性 |
| 600036 | 招商银行 | 银行 | 价值型 |

## 数据时间范围

- **时间跨度**：2019年1月1日 至今（约6年）
- **数据频率**：日度
- **基准指数**：沪深300指数
- **无风险利率**：年化2.5%（日化收益率 = 0.025/252 ≈ 0.0001）

## 分析内容

### 必做部分
1. **数据获取与整理**：收益率计算、描述性统计、正态性检验
2. **全样本Beta系数估计**：基于CAPM模型的OLS回归分析
3. **分年度Beta系数估计**：年度Beta系数变化趋势分析
4. **滚动Beta系数估计**：60日滚动窗口Beta系数动态变化
5. **相关性分析**：个股间相关性及其时变特征
6. **等权重投资组合分析**：组合构建与绩效评价

### 选做部分
7. **有效前沿分析**：基于均值-方差优化的投资组合构建
8. **行业视角讨论**：Beta系数与行业特征的关系分析

## 主要发现

### 1. Beta系数估计结果
- 银行股（平安银行、招商银行）Beta系数相对较低，体现了其防御性特征
- 成长股（东方财富）Beta系数较高，系统性风险较大
- 地产股（万科A）周期性特征明显，Beta在不同时期波动较大

### 2. 时变特征分析
- 2020年疫情期间，大部分股票Beta系数出现显著上升
- 2022年市场调整期，Beta系数普遍波动加剧
- 滚动Beta显示，系统性风险在市场压力期间趋向收敛

### 3. 投资组合效应
- 等权重组合有效降低了整体波动率
- 组合Beta约等于个股Beta的平均值，验证了Beta的可加性
- 分散化投资在降低非系统性风险方面效果显著

### 4. 相关性特征
- 市场剧烈波动期间，股票间相关性显著上升
- 同行业股票相关性明显高于跨行业股票
- 危机期间"相关性趋于1"的现象较为明显

### 5. 风险收益特征
- 高Beta股票在牛市中表现更优，但在熊市中跌幅也更大
- 投资组合的夏普比率优于大部分单一股票
- 最大回撤控制效果显著，风险调整后收益提升明显
