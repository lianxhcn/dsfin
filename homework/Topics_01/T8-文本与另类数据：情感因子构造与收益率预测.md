## 文本与另类数据：情感因子构造与收益率预测

### 任务背景

传统量化研究依赖价格、财务等结构化数据，但大量市场相关信息以非结构化文本形式存在——财报 MD&A、新闻标题、分析师报告、社交媒体等。如何从文本中提取有效信号、构造情感因子，并验证其对股票收益率的预测能力，是现代金融数据分析的重要方向。

本题要求你完成从**文本获取 → 情感打分 → 因子构造 → 预测验证**的完整流程。

---

**文本数据**（以下两类选其一，也可结合使用）：

**选项 A：财经新闻标题**
- 使用 `akshare` 获取财经新闻：`ak.stock_news_em(symbol)` 获取个股新闻

### 分析任务

#### 必做部分

**1. 文本获取与预处理**

- 获取文本数据，整理为结构化表格，包含字段：
  - `code`：股票代码
  - `date`：文本发布日期（精确到日或月）
  - `text`：原始文本内容
- 对文本进行预处理：
  - 去除 HTML 标签、特殊符号、数字（视分析目的而定）
  - 使用 `jieba` 进行中文分词
  - 去除停用词（建议使用哈工大停用词表，可从 GitHub 搜索下载）
- 统计词频，绘制词云图（使用 `wordcloud` 库），直观展示高频词汇

**2. 情感打分：词典法**

使用中文金融情感词典对文本进行情感打分。推荐词典：

- **NTUSD**（台湾大学情感词典，含正负面词汇）
- **SnowNLP** 内置情感模型：`from snownlp import SnowNLP`
- 或自行从 GitHub 搜索"中文金融情感词典"

打分逻辑（以词典法为例）：

```python
def sentiment_score(text, pos_words, neg_words):
    words = jieba.lcut(text)
    pos = sum(1 for w in words if w in pos_words)
    neg = sum(1 for w in words if w in neg_words)
    total = len(words)
    return (pos - neg) / total if total > 0 else 0
```

- 对每条文本计算情感得分，汇总为月度/季度均值，作为**情感因子（Sentiment）**
- 绘制情感因子的时序图，标注重要市场事件，观察是否存在规律性

**3. 情感打分：预训练模型法（选项）**

如有条件，使用 FinBERT 或 ERNIE 等预训练模型进行情感分类（正面/负面/中性）：

```python
# 使用 transformers 库加载 FinBERT
from transformers import pipeline
classifier = pipeline("text-classification",
                      model="ProsusAI/finbert")  # 英文版
# 中文版参考：ERNIE-Finance 或 MacBERT-Finance
```

- 对比词典法和预训练模型法的打分结果，计算两者的相关系数
- 讨论两种方法的优劣及适用场景

**4. 情感因子与收益率的关联分析**

- 计算情感因子与**下期股票超额收益率**的相关系数（同期及滞后 1-3 期）
- 绘制滞后相关系数图，讨论情感因子的预测方向和时效
- 按情感得分高低将样本分为 3 组（正面、中性、负面），计算各组下期平均超额收益率
- 对分组收益率差异进行 t 检验，验证是否显著

**5. 情感因子与传统因子的结合**

- 将情感因子（Sentiment）加入 T3 中的 Fama-MacBeth 截面回归：

$$\text{ExRet}_{i,t+1} = \alpha_t + \beta_{1t} \cdot \text{Sentiment}_{it} + \beta_{2t} \cdot PE_{it} + \beta_{3t} \cdot PB_{it} + \varepsilon_{it}$$

- 比较加入情感因子前后，模型 $R^2$ 的变化，以及情感因子系数的显著性
- 讨论：情感因子是否提供了传统财务因子之外的增量信息？

#### 选做部分

**6. 事件驱动：新闻冲击的短期效应**

- 识别情感得分极端高（前 5%）和极端低（后 5%）的新闻事件
- 以新闻发布日为事件日，计算事件前后 $[-3, +5]$ 日的累计异常收益率（CAR）
- 绘制正面/负面新闻的平均 CAR 对比图，检验非对称性

**7. 情感动量**

- 构造情感动量指标：连续 N 期情感得分均为正面（或负面）的持续性信号
- 检验情感动量是否比单期情感得分具有更强的预测能力
