## A 股市场波动率特征分析与 GARCH 建模

### 任务背景

金融时间序列普遍存在"波动率聚集"（volatility clustering）现象——大波动之后往往跟随大波动，小波动之后往往跟随小波动。GARCH 类模型是刻画和预测条件波动率的标准工具。本题要求你选取若干 A 股指数，系统检验收益率序列的统计特征，构建 GARCH 族模型，并对波动率进行预测评估。

### 分析任务

#### 必做部分

**1. 收益率序列的基本特征检验**

- 计算每个指数的日对数收益率：$r_t = \ln(P_t / P_{t-1})$
- 绘制收益率时序图，直观观察波动率聚集现象
- 进行以下统计检验，汇总为表格：

  | 检验项目 | 方法 | 原假设 |
  |----------|------|--------|
  | 描述性统计 | 均值、标准差、偏度、峰度 | — |
  | 正态性检验 | Jarque-Bera 检验 | 收益率服从正态分布 |
  | 平稳性检验 | ADF 检验 | 序列存在单位根 |
  | 自相关检验 | Ljung-Box 检验（对 $r_t$ 和 $r_t^2$，滞后 10 阶）| 无自相关 |
  | ARCH 效应检验 | ARCH LM 检验（滞后 5 阶）| 无 ARCH 效应 |

- 绘制每个指数收益率的 ACF/PACF 图，同时绘制收益率平方的 ACF/PACF 图
- **结论**：各指数是否存在序列相关？是否存在显著的 ARCH 效应？

**2. GARCH(1,1) 模型估计**

- 使用 `arch` 库对 4 个指数分别估计标准 GARCH(1,1) 模型：

$$r_t = \mu + \varepsilon_t, \quad \varepsilon_t = \sigma_t z_t, \quad z_t \sim N(0,1)$$
$$\sigma_t^2 = \omega + \alpha \varepsilon_{t-1}^2 + \beta \sigma_{t-1}^2$$

- 以表格形式汇报每个指数的参数估计结果：$\hat{\omega}$、$\hat{\alpha}$、$\hat{\beta}$、波动率持续性 $\hat{\alpha} + \hat{\beta}$、对数似然值
- 绘制每个指数的**条件波动率时序图**（年化处理：$\sigma_t^{\text{annual}} = \sigma_t \times \sqrt{252}$）
- 在波动率图中叠加标注以下关键时间节点：
  - 2015 年 A 股股灾（2015-06）
  - 2018 年中美贸易摩擦（2018-10）
  - 2020 年 COVID-19 冲击（2020-02）
  - 2022 年 A 股大幅调整（2022-03）
- 讨论：波动率持续性 $\alpha + \beta$ 在 4 个指数间是否有显著差异？如何解释？

**3. 模型比较**

对每个指数，估计以下三类模型并进行比较：

- **ARCH(q)**：选取适当的 q（建议根据 PACF 图确定，最大不超过 5）
- **GARCH(1,1)**：标准 GARCH
- **GJR-GARCH(1,1)**（又称 TGARCH）：考虑**杠杆效应**，即负向冲击对波动率的影响大于正向冲击：

$$\sigma_t^2 = \omega + (\alpha + \gamma \mathbb{1}_{[\varepsilon_{t-1}<0]}) \varepsilon_{t-1}^2 + \beta \sigma_{t-1}^2$$

- 使用 AIC 和 BIC 进行模型选择，将所有模型的 AIC/BIC 汇总为比较表格
- 报告 GJR-GARCH 中杠杆系数 $\hat{\gamma}$ 的估计值及显著性，讨论 A 股是否存在杠杆效应

**4. 波动率预测与样本外评估**

- 将样本分为：
  - 训练集：前 80%（约 2010-2021 年）
  - 测试集：后 20%（约 2022 年至今）
- 在测试集上，对每个指数使用 GARCH(1,1) 进行**滚动 1 步向前预测**（expanding window 或 rolling window 均可，需说明选择）
- 以**已实现波动率的代理变量**（$|r_t|$ 或 $r_t^2$）作为真实波动率
- 计算预测误差指标：MSE（均方误差）、MAE（平均绝对误差）、QLIKE 损失
- 将 4 个指数的预测精度汇总为表格，讨论不同板块波动率的可预测性差异

#### 选做部分

**5. 多指数波动率联动分析**

- 提取 4 个指数的 GARCH(1,1) 条件波动率序列
- 计算条件波动率之间的相关系数矩阵（静态），以及滚动相关系数（动态）
- 讨论：不同板块（大盘蓝筹 vs 成长科技）的风险联动特征

**6. 基于 GARCH 的 VaR 计算与回测**

- 基于每日条件波动率，计算每日 99% 在险价值（VaR）：

$$\text{VaR}_{t} = \hat{\mu} + z_{0.01} \cdot \hat{\sigma}_t$$

  其中 $z_{0.01} = -2.326$（正态分布 1% 分位数）

- 统计测试集内 VaR 超越次数，与理论超越频率（1%）进行比较
- 使用 Kupiec 检验（失败率检验）验证 VaR 模型的有效性
