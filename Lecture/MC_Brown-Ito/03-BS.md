# 从布朗运动到 Black–Scholes：风险中性与欧式看涨定价

上一节我们停在几何布朗运动和 Ito 引理。这一节往前走一步：
在“随机游走”的世界里，期权到底是怎么定价的？这时就要引入两个关键概念：

* 风险中性定价
* Black–Scholes 公式

下面还是按照由浅入深的顺序来讲。

---

## 1. 两个“世界”：真实世界 P 与风险中性世界 Q

### 1.1 真实世界 P：收益里有风险补偿

在最简单的设定下，我们可以假设股票价格在真实世界 P 下满足几何布朗运动：

$$dS_t = \mu S_t dt + \sigma S_t dW_t$$

其中：

* $\mu$：真实世界下的期望收益率，包含风险溢价
* $\sigma$：波动率
* $W_t$：真实世界 P 下的布朗运动

真实世界里的投资者是“厌恶风险”的，他们要拿到高于无风险利率 $r$ 的期望收益率 $\mu$，才能愿意持有风险资产。

### 1.2 风险中性世界 Q：所有资产的“期望收益率 = r”

风险中性世界 Q 是一个“虚构”的概率世界，有两个核心特点：

1. 所有资产的“贴现价格”都是鞅：
   $\frac{S_t}{B_t}$ 在 Q 下是鞅，其中 $B_t = e^{rt}$ 是无风险资产价格。

2. 换句话说，在 Q 下任何资产的“期望收益率”都是无风险利率 $r$，不再显式出现风险溢价。

在 Q 世界中，股票价格的动态会变成：

$$dS_t = r S_t dt + \sigma S_t dW_t^Q$$

这里：

* $W_t^Q$：在 $Q$ 下的布朗运动
* 漂移变成 $r$，波动率 $\sigma$ 保持不变

你可以把“从 P 到 Q”理解为：

> 用一个合适的“概率权重调整”（Radon–Nikodym 导数），
> 把原本包含风险溢价的过程，变成一个“期望收益率等于无风险利率”的过程。

这个过程本身是数学上的“测度变换”，但对金融学使用者来说，最重要的结论只有一句话：

> 在 Q 下，贴现后的资产价格是鞅，因此定价公式可以写成
> $\text{价格} = \text{贴现后的风险中性期望}$。

---

## 2. 风险中性定价公式：核心一句话

对于在时点 0 定价、到期 T 的任何衍生品，其在 Q 下的定价公式为：

$$V_0 = e^{-rT}\mathbb{E}^Q[\text{到期时的现金流}]$$

对一份到期 T 的欧式看涨期权，执行价为 $K$，到期支付为：

$$(S_T - K)^+ = \max(S_T - K,0)$$

因此它的定价公式是：

$$C_0 = e^{-rT}\mathbb{E}^Q[(S_T - K)^+]$$

所以，问题被化简为两个步骤：

1. 在风险中性世界 Q 中写出 $S_T$ 的分布；
2. 对 $\max(S_T - K,0)$ 做风险中性期望，再贴现。

---

## 3. 几何布朗运动在 Q 世界中的分布

在 Q 下，几何布朗运动满足：

$$dS_t = r S_t dt + \sigma S_t dW_t^Q$$

和上一节一样，我们令 $X_t = \ln S_t$，用 Ito 引理：

$$dX_t = \left(r - \tfrac12\sigma^2\right)dt + \sigma dW_t^Q$$

积分得到：

$$X_T = \ln S_T = \ln S_0 + \left(r - \tfrac12\sigma^2\right)T + \sigma W_T^Q$$

由于 $W_T^Q\sim N(0,T)$，可知：

$$\ln S_T\sim N\left(\ln S_0 + \left(r - \tfrac12\sigma^2\right)T,\ \sigma^2 T\right)$$

因此 $S_T$ 服从对数正态分布。

这个结论是 Black–Scholes 推导中的关键：
我们知道了在 Q 世界下，$$S_T$ 的概率分布完全由 $S_0,r,\sigma,T$ 四个参数决定。

## 4. 推导 Black–Scholes 公式：拆成两项期望

### 4.1 把期权支付拆开

期权价值：

$$C_0 = e^{-rT}\mathbb{E}^Q[(S_T - K)^+] = e^{-rT}\mathbb{E}^Q[(S_T - K)1_{{S_T>K}}]$$

拆成两项：

$$
\begin{aligned}
C_0
&= e^{-rT}\left(\mathbb{E}^Q[S_T 1_{{S_T>K}}] - K\mathbb{E}^Q[1_{{S_T>K}}]\right)\
&= e^{-rT}\mathbb{E}^Q[S_T 1_{{S_T>K}}] - K e^{-rT}Q(S_T>K)
\end{aligned}
$$

记：

* 第一项：$A = e^{-rT}\mathbb{E}^Q[S_T 1_{{S_T>K}}]$

* 第二项：$B = K e^{-rT}Q(S_T>K)$

我们分别求 $A$ 和 $B$。

### 4.2 简化分布：用标准正态变量表示

设：

$$\ln S_T = \ln S_0 + \left(r - \tfrac12\sigma^2\right)T + \sigma\sqrt{T}Z$$

其中 $Z\sim N(0,1)$。

于是：

$$S_T = S_0\exp\left(\left(r - \tfrac12\sigma^2\right)T + \sigma\sqrt{T}Z\right)$$

不等式 $S_T>K$ 等价于：

$$
\ln S_T > \ln K \
\Rightarrow \ln S_0 + \left(r - \tfrac12\sigma^2\right)T + \sigma\sqrt{T}Z > \ln K
$$

解出 $Z$：

$$
Z > \frac{\ln(K/S_0) - \left(r - \tfrac12\sigma^2\right)T}{\sigma\sqrt{T}}
$$

定义：

$$
d_2 = \frac{\ln(S_0/K) + \left(r - \tfrac12\sigma^2\right)T}{\sigma\sqrt{T}},\quad
d_1 = d_2 + \sigma\sqrt{T}
$$

注意这里 $Z>S$ 的阈值可以写成 $Z > -d_2$，于是：

$$Q(S_T>K) = Q(Z>d_2^*) = 1-\Phi(d_2^*) = \Phi(d_2)$$

经过标准化，我们可以得到：

* $Q(S_T>K) = \Phi(d_2)$$
* $\mathbb{E}^Q[S_T 1_{{S_T>K}}] = S_0\Phi(d_1)$$

这两个结果是通过对对数正态分布做积分、再换元为标准正态分布得到的。推导细节如果写得很完整会比较长，但最后的结论非常简洁。

于是：

* $A = e^{-rT}\mathbb{E}^Q[S_T 1_{{S_T>K}}] = e^{-rT}S_0\Phi(d_1)$$
* $B = K e^{-rT}Q(S_T>K) = K e^{-rT}\Phi(d_2)$$

把 A、B 代回期权价格：

$$
C_0 = S_0\Phi(d_1) - K e^{-rT}\Phi(d_2)
$$

这就是著名的 Black–Scholes 欧式看涨期权定价公式。

归纳一下：

* 期权价格是“今天的股票价格 × 上涨到执行价之上的概率权重”减去“贴现后的执行价 × 到期执行概率权重”；
* 两个概率权重分别是 $\Phi(d_1)$ 和 $\Phi(d_2)$。

## 5. 经济学直觉：为什么可以用“风险中性期望”？

上面推导看起来有点“数学化”：变换测度、布朗运动、积分求期望。但经济学直觉其实可以用一句话概括：

> 如果市场没有套利，那么任何衍生品的价格，
> 都可以用它未来现金流在某个“风险中性世界”下的期望贴现来表示。

背后的逻辑是：

1. 在 Black–Scholes 的框架下，可以用股票和无风险资产动态复制期权；
2. 如果期权价格偏离复制组合的成本，就会产生套利机会；
3. 因此，期权价格必须等于复制组合的成本；
4. 这个复制组合的成本，恰好可以写成“风险中性期望贴现”的形式。

也就是说：

* 风险中性世界 Q 并不是“真实世界”，而是一个便于定价的数学世界；
* 在这个世界里，所有资产的期望收益率等于无风险利率；
* 在这个世界下做期望，再贴现，就可以得到没有套利的合理价格。

## 6. 一个具体数值例子

设 $S_0 = 100$，$K = 100$，$r = 5\%$，$\sigma = 20\%$，$T = 1$ 年，则：

$$
d_1 = \frac{\ln(S_0/K) + (r + \tfrac12\sigma^2)T}{\sigma\sqrt{T}}
= \frac{0 + (0.05 + 0.5\times0.2^2)\times1}{0.2}
= \frac{0.07}{0.2} = 0.35
$$

$$
d_2 = d_1 - \sigma\sqrt{T} = 0.35 - 0.2 = 0.15
$$

查标准正态分布表（或用软件）可得，$\Phi(0.35)\approx 0.6368$，$\Phi(0.15)\approx 0.5596$。于是：

$$
C_0 = S_0\Phi(d_1) - K e^{-rT}\Phi(d_2)
= 100\times0.6368 - 100\times e^{-0.05}\times0.5596
$$

计算：

* $e^{-0.05}\approx 0.9512$$
* 第二项约为 $100\times0.9512\times0.5596\approx 53.25$$
* 第一项为 $63.68$$

所以：

$$
C_0\approx 63.68 - 53.25 \approx 10.4
$$

这就是在该参数下，欧式看涨期权的 Black–Scholes 理论价格，单位与价格一致（例如美元）。

## 7. 用 Stata 做一个风险中性 Monte Carlo，验证 Black–Scholes

下面给一段 Stata 代码，用 Monte Carlo 模拟检验 Black–Scholes 公式：

* 在风险中性世界 Q 下，模拟很多条几何布朗运动路径；
* 计算每条路径下的到期支付 $\max(S_T-K,0)$；
* 打折并取平均，得到 Monte Carlo 估计价；
* 与理论公式 $C_0 = S_0\Phi(d_1) - K e^{-rT}\Phi(d_2)$ 对比。

```stata
*======================================================
* Monte Carlo 模拟验证 Black–Scholes 看涨期权定价
*======================================================

clear all
set more off

*-----------------------------
* 1. 设定参数
*-----------------------------
local S0    = 100      // 初始价格 S_0
local K     = 100      // 执行价 K
local r     = 0.05     // 无风险利率 r
local sigma = 0.2      // 波动率 σ
local T     = 1        // 到期时间 T (年)

local Nstep = 252      // 每年交易日数
local Npath = 50000    // Monte Carlo 路径数，可适当调整

local dt = `T'/`Nstep'

*-----------------------------
* 2. 生成路径 × 时间 的数据结构
*-----------------------------
set obs `Npath'
gen id = _n                          // 路径编号

expand `=`Nstep'+1''
bysort id: gen t_index = _n - 1      // 0,...,Nstep
gen t = t_index*`dt'                 // 对应时间

*-----------------------------
* 3. 在 Q 世界下生成布朗运动增量 dW^Q
*-----------------------------
set seed 20251107
gen dW = rnormal(0, sqrt(`dt'))

* 每条路径在 t=0 时的增量设为 0
bysort id (t): replace dW = 0 if t_index == 0

*-----------------------------
* 4. 递推几何布朗运动：dS = r S dt + σ S dW^Q
*   显式解为：
*   S_{t+dt} = S_t * exp((r - 0.5σ^2)dt + σ dW)
*-----------------------------
gen S = .

bysort id (t): replace S = `S0' if t_index == 0

forvalues k=1/`Nstep'{
    quietly replace S = S[_n-1]* ///
        exp((`r' - 0.5*`sigma'^2)*`dt' ///
            + `sigma'*dW) ///
        if t_index == `k'
}

*-----------------------------
* 5. 取到期 T 时刻的价格，计算期权支付
*-----------------------------
preserve
keep if t_index == `Nstep'

gen payoff = max(S - `K', 0)
gen disc_payoff = exp(-`r'*`T')*payoff

sum disc_payoff
display "Monte Carlo 估计的期权价格 C0_MC = " r(mean)

*-----------------------------
* 6. 计算 Black–Scholes 理论价格
*-----------------------------
scalar d1 = (ln(`S0'/`K') + (`r' + 0.5*`sigma'^2)*`T')/(`sigma'*sqrt(`T'))
scalar d2 = d1 - `sigma'*sqrt(`T')

* 正态分布函数：使用 Stata 内置函数 normal()
scalar Phi_d1 = normal(d1)
scalar Phi_d2 = normal(d2)

scalar C0_BS = `S0'*Phi_d1 - `K'*exp(-`r'*`T')*Phi_d2

display "Black–Scholes 理论价格 C0_BS = " C0_BS

restore
```

运行后，你会看到两行关键输出：

* Monte Carlo 估计的价格 `C0_MC`
* Black–Scholes 理论价格 `C0_BS`

当 `Npath` 足够大时，这两个数应该非常接近，这就是“风险中性定价 + 几何布朗运动”这一整套框架在数值上的自洽性体现。

## 8. 小结与下一步方向

到这里，我们串起来了这样一条线：

1. 用布朗运动描述连续时间的随机扰动；
2. 用几何布朗运动建模股票价格；
3. 通过 Ito 引理掌握价格与对数价格之间的转换；
4. 在风险中性世界 Q 下写出 $S_T$ 分布；
