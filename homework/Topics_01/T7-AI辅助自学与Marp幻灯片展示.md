## 研究工具训练（下）：AI 辅助自学与 Marp 幻灯片展示

### 任务背景

数据科学领域的方法更新极快，依赖课堂讲授远远不够。本题模拟一个真实的研究场景：你在文献中遇到了一个陌生的方法，需要通过自主检索和与 AI 对话快速上手，最终整理成可以向他人讲解的笔记和幻灯片。

这道题考察的不是你最终掌握了多少，而是**你如何学**——对话过程的真实性、追问的深度、以及将零散对话整理为结构化文档的能力。

> **特别说明**：AI 对话记录是评分的核心依据。评分时将重点审查对话的递进性——困惑、追问、纠错、验证——而非一次性生成答案后倒推提问记录。请保留真实的探索轨迹。

### 任务二：AI 辅助自学——对话记录

#### 2.1 对话要求

与 AI（推荐 Claude、ChatGPT 或 Kimi）进行不少于 **15 轮**的结构化对话，逐步深入理解所选方法。

**核心原则**：每一轮提问都应基于上一轮的回答——追问不清楚的地方，要求举例，验证代码，质疑假设。不允许一次性发出"请写一篇关于 XXX 的完整教程"这样的提示词。

以下是以 **IPW** 为例的推进路线，供参考（其他主题请自行设计类似的递进路径）：

```
第 1 轮  请用最简单的语言解释 IPW 是什么，它要解决什么问题？
第 2 轮  你提到"选择偏误"，能用一个金融领域的具体例子说明吗？
第 3 轮  倾向得分（propensity score）是什么？用什么模型来估计它？
第 4 轮  得到倾向得分之后，权重具体是怎么计算的？写出公式。
第 5 轮  这个公式里分母接近 0 会有什么问题？实践中怎么处理？
第 6 轮  请用 Python 写一个最小可运行的 IPW 示例，用模拟数据就行。
第 7 轮  你代码里第 X 行的 XXX 是什么意思？为什么不用 YYY？
第 8 轮  运行你的代码后我得到了 ZZZ 报错，怎么解决？
第 9 轮  IPW 和倾向得分匹配（PSM）有什么本质区别？各适合什么场景？
第 10 轮 IPW 依赖哪些核心假设？如果假设不满足会怎样？
第 11 轮 有没有改进版本的 IPW？比如双重稳健估计量是什么？
第 12 轮 在 A 股研究中，IPW 通常用来研究什么问题？能举一个例子吗？
第 13 轮 你提到的那篇论文我查不到，能给我一个可以验证的来源吗？
第 14 轮 如果我要用 IPW 研究"机构投资者持股对公司信息披露质量的影响"，
         数据该怎么构造？处理变量和结果变量分别是什么？
第 15 轮 帮我总结一下 IPW 的完整学习路径，以及我还需要补充哪些前置知识？
```

**"我的评注"是必填项**，哪怕只有一句话，例如：

- "AI 给的代码跑出了报错，需要调试"

### 任务四：制作 Marp 幻灯片并展示

#### 4.1 Marp 简介与安装

[Marp](https://marp.app) 是一个基于 Markdown 的幻灯片工具。你用写 `.md` 文件的方式制作幻灯片，可以直接和代码、笔记放在同一个 GitHub 仓库中统一管理，便于版本追踪。

安装方式（任选其一）：

- **VS Code 插件**（推荐）：在 VS Code 扩展商店搜索 `Marp for VS Code` 安装，可实时预览
- **命令行**：`npm install -g @marp-team/marp-cli`

#### 4.2 制作幻灯片

基于任务三的笔记，制作一份 Marp 幻灯片，保存为 `slides/slide_[主题名].md`，**共 10-12 张**，结构如下：

| 张数 | 内容 |
|------|------|
| 第 1 张 | 封面：主题名称、小组成员、日期 |
| 第 2 张 | 目录 |
| 第 3 张 | 动机：这个方法解决什么问题？直接分析会犯什么错误？ |
| 第 4-5 张 | 方法原理：步骤 + 关键公式 |
| 第 6 张 | Python 代码示例（关键片段，无需完整） |
| 第 7 张 | 金融应用场景（2 个例子） |
| 第 8 张 | 与相关方法的对比（表格） |
| 第 9 张 | 局限性 |
| **第 10 张** | **学习反思：AI 哪里不准确？走了哪些弯路？** |
| 第 11 张 | 参考资料 |

**Marp 幻灯片文件头部模板**：

```markdown
---
marp: true
theme: default
paginate: true
math: katex
---

# IPW：逆概率加权
## 金融数据分析与建模 · 小组作业 T07

**小组成员**：张三、李四、王五  
**日期**：2025 年 XX 月

---

## 目录

1. 动机与问题
2. 方法原理
3. Python 实现
4. 金融应用
5. 方法对比
6. 局限性
7. 学习反思

---

## 动机：直接比较会犯什么错误？

> 研究问题：机构投资者持股↑ → 公司信息披露质量↑ ?

- 直接回归：持股比例高的公司披露质量也高
- **问题**：大公司往往同时拥有更多机构持股 **和** 更好的披露质量
- 混淆变量（公司规模）导致因果推断失效

$$\hat{\tau}_{\text{naive}} = E[Y|D=1] - E[Y|D=0] \neq \text{ATE}$$

---
```

#### 4.3 导出为 PDF

使用以下命令将幻灯片导出为 PDF（需已安装命令行工具），将 PDF 文件存放至 `output/` 文件夹：

```bash
marp slides/slide_topic.md --pdf -o output/slide_topic.pdf
```

如使用 VS Code 插件，可在编辑器右上角点击"Export slide deck"直接导出。
