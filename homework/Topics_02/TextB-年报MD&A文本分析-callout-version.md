## 文本分析作业（B）：上市公司年报 MD&A 文本分析

> **作业性质**：小组作业（建议 3-4 人）  
> **完成时间**：两周  
> **工具**：Python，Jupyter Notebook  
> **提交方式**：GitHub 仓库链接 + Quarto Book（GitHub Pages）

---

### 研究背景

上市公司年报中的"管理层讨论与分析"（Management Discussion & Analysis，MD&A）是信息披露质量研究的核心文本来源之一。与财务报表的数字不同，MD&A 用自然语言描述公司的经营状况、风险因素和未来展望，是管理层向投资者传递软信息的主要渠道。

学术研究表明，MD&A 的语调（乐观/悲观）、可读性（复杂度）、内容相似度（与上年相比是否有实质变化）都对股价反应和分析师预测具有增量预测能力（Tetlock et al., 2008; Li, 2010; Loughran & McDonald, 2011）。

本作业选取 **1 个行业**的 **8-12 家**上市公司，分析其近 **5 年**（2019-2023）的 MD&A 文本，综合运用文本清洗、情感分析、可读性分析、主题建模等方法，检验 MD&A 文本特征是否与公司财务表现和股价变动存在关联。

### 行业与样本选择

**选取一个行业**，要求：

- 样本公司数量：8-12 家，确保行业内有足够的横截面比较
- 选取原则：公司规模不宜差异过大，最好是同质性较高的子行业（如同为整车制造，不要把整车和零部件混在一起）
- 剔除标准：研究期间内有 ST 记录的公司建议剔除（财务数据异常）

**建议行业**（可自选其他）：

| 行业 | 特点 | 参考公司 |
|------|------|---------|
| 银行（城商行/股份行） | MD&A 格式标准，风险披露丰富 | 招商、兴业、浦发、民生等 |
| 新能源（光伏/风电） | 近年政策变化快，MD&A 语调变化明显 | 隆基、通威、晶科等 |
| 医药（创新药） | 研发进展披露复杂，专业术语多 | 恒瑞、百济、药明等 |
| 消费（食品饮料） | 语言相对易读，便于可读性分析 | 茅台、五粮液、伊利等 |

### 第一部分：数据获取

**年报来源**：巨潮资讯网（`www.cninfo.com.cn`）

年报 PDF 文件可通过巨潮资讯的 API 接口批量下载，无需手动逐一下载。

**建议使用的包**：`requests`、`pdfplumber`（或 `PyMuPDF`/`fitz`）、`pandas`、`re`、`pathlib`

::: {.callout-tip}
### 提示词框：批量下载年报 PDF
给 AI 的提示词应包含以下信息：

1. 目标网站：巨潮资讯（cninfo.com.cn）
2. 你需要的年报类型：年度报告（区别于半年报、季报）
3. 公司列表：股票代码列表（如 `["600036", "601166", ...]`）
4. 年份范围：2019-2023
5. 要求 AI 先分析巨潮的公告查询 API 接口（建议用 DevTools 自行抓包找到接口 URL 和参数），然后编写批量下载函数，将 PDF 按 `{股票代码}_{年份}.pdf` 的命名规则保存到本地
6. 要求加入下载日志（成功/失败/文件大小），以及断点续传逻辑（已下载的文件跳过）
:::

::: {.callout-warning}
### 避坑指南：PDF 下载常见问题
- 巨潮的公告 API 返回的是公告列表，同一公司同一年可能有多个"年度报告"（如修订版、英文版），需要用关键词过滤（标题包含"年度报告"且不包含"摘要"、"英文"、"修订"）
- 部分公司年报是扫描件（图片型 PDF），文字提取会完全失败——建议先检查提取结果字符数，字符数极少的 PDF 可能是扫描件，需单独处理或排除
- PDF 文件体积可能较大（单份 10-50MB），批量下载前估计总体积，注意磁盘空间
:::

### 第二部分：MD&A 章节提取

年报中的 MD&A 通常位于"第四节 经营情况讨论与分析"或类似标题，需要从完整年报 PDF 中精确提取该章节。

**建议使用的包**：`pdfplumber`、`re`、`pandas`

::: {.callout-tip}
### 提示词框：章节定位提取
MD&A 提取是本作业技术难度最高的环节。给 AI 的提示词应说明：

1. 目标章节的常见标题变体（如"经营情况讨论与分析"、"管理层讨论与分析"、"董事会报告"等）
2. 章节结束的判断依据（下一节标题出现时停止提取）
3. 你希望的处理逻辑：先提取全文，再用正则匹配章节起止位置
4. 要求 AI 在提取完成后，打印每份年报提取到的文本前 200 字，供人工验证是否提取正确
5. 对于提取失败的文件，要求记录到日志，不要静默跳过

重要提示：让 AI 解释为什么不建议用固定页码来定位章节（年报结构每年可能略有不同），以及基于标题文本匹配为何更鲁棒。
:::

::: {.callout-warning}
### 避坑指南：章节提取质量控制
- 不同公司、不同年份的年报章节编号不统一：有的是"第四节"，有的是"第三节"，有的直接是"三、"。提取规则需要覆盖多种变体，否则部分年报会提取失败
- pdfplumber 对双栏排版的处理较差（文字顺序错乱），若发现大量文本乱序，尝试改用 PyMuPDF（`fitz`）
- 提取完成后务必检查：每份 MD&A 的字符数是否在合理范围（通常 5000-50000 字）？字符数异常少的可能提取失败，异常多的可能提取了超出章节的内容
:::

完成提取后，将结构化数据存为一个面板数据集，包含字段：`股票代码`、`公司名称`、`年份`、`MD&A全文`，后续所有分析均基于此数据集。

### 第三部分：文本预处理

**建议使用的包**：`jieba`、`re`、`pandas`

预处理步骤：

1. **清洗**：去除页码、表格残留数字、乱码字符、重复段落（年报有时会重复同一段落）
2. **分词**：使用 `jieba`，加载金融+行业专业词典
3. **去停用词**：加载通用停用词表，并额外删除年报套话（如"本公司"、"报告期内"、"如下"等）

::: {.callout-tip}
### 提示词框：年报专用停用词
年报文本有特有的无信息词汇，通用停用词表无法覆盖。提示 AI 时说明：

1. 你的行业
2. 要求 AI 生成一份 100-200 词的年报专用停用词列表，包含：公文套话、格式标记词、高频但无语义的词（如"本报告"、"截至"、"详见"）
3. 要求 AI 解释：为什么年报中的公司名称本身也应该加入停用词（所有公司的年报都会大量提到自身名称，反而成为噪声）
:::

### 第四部分：情感分析

**建议使用的包**：`pandas`、`re`、`matplotlib`/`plotly`

本作业要求使用**金融领域情感词典**（而非通用词典），原因是 MD&A 中大量词汇的情感含义与日常语言不同（如"风险"在日常语言是负面词，但在年报中出现频率极高，未必代表负面情绪）。

**推荐词典**：Loughran-McDonald 金融词典的中文版（可在 GitHub 搜索"Loughran McDonald 中文"获取），包含正面词、负面词、不确定词、模态词等多个类别。

::: {.callout-tip}
### 提示词框：MD&A 情感评分
给 AI 的提示词应包含：

1. 你使用的词典格式（正面词列表、负面词列表）
2. 你希望的情感指标定义（如：情感净值 = (正面词数 − 负面词数) / 总词数）
3. 要求 AI 额外计算**不确定性词比例**（Uncertainty），这在 MD&A 研究中是独立的重要维度（高不确定性预示公司面临较多未知风险）
4. 要求 AI 将情感分数与公司财务数据（如当年 ROA、下一年股价收益率）进行散点图可视化，初步观察关联
5. 要求 AI 解释词典法的局限性，以及与机器学习情感模型相比的优劣
:::

**分析输出**：

- 各公司各年的情感得分面板数据（`公司 × 年份`热力图）
- 情感得分与同期 ROA 的散点图（横截面，按年分面）
- 情感得分的时序趋势（各公司折线图，标注行业重大事件年份）

::: {.callout-warning}
### 避坑指南：情感分析结果解读
- MD&A 的情感得分通常整体偏正面（管理层天然倾向于积极表述）。更有意义的分析是年度变化（本年比上年更悲观了吗？）和公司间相对比较（哪家公司语调最谨慎？），而不是绝对值的高低
- 公司规模不同会影响词频统计，建议使用比例而非绝对词数
:::

### 第五部分：可读性分析

可读性（Readability）是 MD&A 研究中的独特维度。研究表明，业绩较差的公司往往倾向于用更复杂、更难懂的语言来掩盖负面信息（Bloomfield, 2002 的"不完整揭示假说"）。

**建议使用的包**：`textstat`（英文）、自行实现中文可读性指标

**中文可读性指标**（选择 1-2 个计算）：

| 指标 | 定义 | 解读 |
|------|------|------|
| 平均句长 | 总字数 / 句子数 | 句子越长越难读 |
| 平均词长 | 总字数 / 词数 | 词汇越复杂越难读 |
| 词汇丰富度（TTR） | 不重复词数 / 总词数 | 值越高说明用词越多样（正向） |
| 专业术语比例 | 专业词汇数 / 总词数 | 行业内投资者关注，外部投资者难以理解 |

::: {.callout-tip}
### 提示词框：可读性指标计算
给 AI 的提示词应说明：

1. 你选择的 1-2 个可读性指标及其定义
2. 输入数据格式（已分词的词列表，或原始文本）
3. 要求 AI 编写计算函数，并解释每个指标的经济直觉
4. 要求 AI 分析：可读性是否随时间趋势下降？（年报普遍越写越长、越写越复杂）；可读性差的公司下一年 ROA 是否更低？
:::

### 第六部分：主题建模（LDA）

**建议使用的包**：`gensim`、`pyLDAvis`

对 MD&A 语料进行主题建模，目标是识别行业内公司共同关注的核心议题，以及不同公司在议题上的侧重差异。

::: {.callout-tip}
### 提示词框：跨年度主题演化
MD&A 的主题建模有一个特别有价值的分析维度：**主题比例随时间的变化**。提示 AI 时说明：

1. 你希望建立一个跨公司、跨年度的统一 LDA 模型（而非分年度建模）
2. 提取每个文档（公司-年份）的主题权重后，按年度聚合，绘制主题比例的时序折线图
3. 要求 AI 解释：为什么统一模型比分年度模型更适合做时序比较（保证主题定义的一致性）
4. 要求 AI 帮你给每个主题命名（基于 Top 15 关键词），并说明命名的依据
:::

**特别分析：文本相似度**

计算每家公司相邻年份 MD&A 的文本相似度（余弦相似度），衡量"信息更新程度"：

::: {.callout-tip}
### 提示词框：年报文本相似度
给 AI 的提示词应说明：

1. 计算思路：对每家公司，计算第 t 年和第 t-1 年 MD&A 的 TF-IDF 向量，然后计算余弦相似度
2. 相似度越高（接近 1）意味着两年年报几乎一样，即"复制粘贴"式年报，信息含量低
3. 要求 AI 绘制各公司的相似度时序图，以及相似度与下一年股价收益率的散点图
4. 讨论问题：相似度极高（>0.95）的年报是否应该视为信息披露质量差的信号？
:::

### 第七部分：文本特征与财务/股价的关联分析

**建议使用的包**：`pandas`、`statsmodels`、`plotly`

将前几部分构造的文本特征（情感得分、不确定性词比例、可读性指标、主题比例、文本相似度）与公司财务数据和股价数据合并，构建面板数据集：

- 财务数据：从 CSMAR 下载同期 ROA、ROE、资产负债率（与资本结构作业数据可直接复用）
- 股价数据：使用 akshare 获取年度收益率

**分析框架**：

1. **描述性证据**：对每个文本特征，绘制其与下一年 ROA 的散点图（按公司着色），观察是否有规律
2. **相关系数矩阵**：计算所有文本特征和财务指标之间的 Pearson 相关系数，绘制热力图
3. **面板回归（选做）**：以下一年 ROA 或股价收益率为因变量，文本特征为解释变量，控制公司和年度固定效应，使用 `pyfixest` 或 `linearmodels` 估计

::: {.callout-warning}
### 避坑指南：文本-财务关联分析
- 面板数据的时间维度只有 5 年（2019-2023），样本量小，统计检验力不足，所有显著性结论都要谨慎表述
- 文本特征与财务指标之间可能存在反向因果：差公司倾向于用负面语气，但用负面语气不一定导致差业绩。在讨论部分要明确说明这一局限性
- 股价收益率受系统性风险影响大，建议使用超额收益率（相对沪深 300）代替原始收益率，以减少市场因素干扰
:::
