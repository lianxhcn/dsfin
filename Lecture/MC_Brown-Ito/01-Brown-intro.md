## 看不见的“微小冲击”叠加起来的乱走

### 花粉的随机运动

1827 年植物学家布朗观察到：用显微镜观察水中的花粉，会发现这些小颗粒在水中无规则地抖动、乱走，既不是直线也不是有固定方向。

后来爱因斯坦解释为：花粉颗粒不断受到周围大量水分子的随机碰撞，每个方向的撞击力随机且不平衡；这些微小的撞击差累积起来，就形成了肉眼可见的无规则运动。这种由大量微观随机因素叠加导致的宏观随机现象，就是布朗运动的本质。

如果用一条曲线画出 “花粉位置随时间的变化”，你会发现它具有如下特征：

- **随机性**：它的未来路径是不可预测的。
- **无规则**：它的运动没有固定的方向或模式。
- **连续但不可微**：它的轨迹是连续的（没有跳跃），但处处"尖刺"，无法求导（速度在无限小的间隔内都会改变方向）。

### 金融里的类比：资产价格的“随机扰动”

我们可以把那个在水中跳舞的花粉颗粒，想象成**一只股票的价格**。

-   **水分子的随机碰撞** &rarr; **市场上无数交易者的买卖行为**。每个交易者基于自己的信息、情绪和判断做出决策，这些决策汇总起来，形成了对股票价格的"随机冲击"。好消息和买盘像一股"撞击力"推高价格，坏消息和卖盘则像另一股"撞击力"打压价格。

-   **花粉的无规则运动路径** &rarr; **股票价格的随机波动路径**。你无法准确预测下一分钟股票价格是涨是跌，就像你无法预测花粉下一瞬间会往哪边运动一样。

因此，在很多连续时间金融模型中，会用布朗运动来刻画“连续不断的小波动”。


## 数学定义：布朗运动（维纳过程）

在数学和金融理论里，最常用的是一维标准布朗运动，通常记为随机过程 $W_t$。具体而言，$(W_t)_{t\ge0}$ 是一个满足下列条件的随机过程：

1. **起点为零**：
   
   $$W_0=0$$

2. **增量服从正态分布且均值为零（平稳增量）**：
   对任意 $0\le s<t$，有

   $$W_t-W_s\sim N(0,t-s)$$
   
   也就是说，在时间间隔 $t-s$ 内，增量 $W_t-W_s$ 服从均值为 $0$，方差为 $t-s$ 的正态分布。如果我们把一个固定的时间步长记为 $h>0$，则可以写成
   
   $$W_{t+h}-W_t\sim N(0,h)$$

   由上式可以看出，增量的分布只与时间间隔 ($h$) 有关，而与时点位置 ($t$) 无关。

3. **增量独立**：
   对任意几段互不重叠的时间区间，比如 $[0,1]$、$[1,2]$、$[5,7]$，对应的增量 $W_1-W_0$、$W_2-W_1$、$W_7-W_5$ 相互独立。

4. **轨迹连续但几乎处处不可导**：
   当时间 $t$ 连续变化时，布朗运动在各个时刻的取值 $W_t$ 也连续变化，中间不会突然跳跃。直观地说，如果时间 $t$ 只发生很小的变动，对应的 $W_t$ 也只会发生很小的变化。然而，它在数学上几乎处处不可导，会呈现出很多 “尖刺” 和 “锯齿”，看起来非常粗糙。

这四点合在一起，就是布朗运动（也称“维纳过程”）的标准定义。

从更直观的角度，可以把布朗运动粗略理解为：

* 把时间轴切成很多很短的小段，每一小段都是在此前的基础上加上一点服从正态分布的随机增量，而且不同小段上的增量彼此独立；
* 把这些独立的小增量按时间顺序累加起来，就得到一条没有断点、但始终在细微抖动的随机曲线，这就是布朗运动的一条样本路径。

下面这一段 Stata 代码，可以帮助你模拟出布朗运动的样本路径：

```stata
clear all
set obs 1000
gen t = _n - 1
gen h = 1
gen dW = rnormal(0, sqrt(h))  // 生成增量，服从 N(0,h)
gen W = sum(dW)               // 累加增量，得到布朗运动
twoway line W t, title("布朗运动样本路径") ///
    ytitle("W_t") xtitle("时间 t")
```

二维布朗运动（二维维纳过程）则是由两个独立的一维布朗运动组成，通常记为 $(W_x(t), W_y(t))$。它们分别表示在 $x$ 轴和 $y$ 轴方向上的布朗运动。



## 3. 几个重要性质

### 3.1 任意时刻的分布

由上面的定义可以推出，对于任意 $t\ge0$：

$$W_t\sim N(0,t)$$

这是因为 $W_t=W_t-W_0$，根据增量的正态性和 $W_0=0$，立刻得到上式。
这句话的直观含义是：时间越长，总体波动的方差越大，布朗运动的轨迹在数轴上“散开得越厉害”。

### 3.2 缩放性质（自相似）

布朗运动有一个非常重要的“自相似”性质：对任意常数 $c>0$，且 $t \ge 0$，有

$$W_{ct} \overset{d}{=} \sqrt{c} W_t$$

该式的含义是：当我们把考察的时间长度放大为原来的 $c$ 倍时，用标准差度量的波动幅度会按 $\sqrt{c}$ 的比例放大。例如，把考察的时间长度从 1 个月扩展到 1 年（此时 $c=12$），则 1 年期收益的标准差大约是 1 个月期收益标准差的 $\sqrt{12}$ 倍。

这和金融中“日收益、周收益、月收益”之间的相似性有什么关系？

在经典几何布朗运动模型里，资产对数价格通常写成
 
$$\log S_t=\log S_0+\mu t+\sigma W_t$$

于是长度为 $h$ 的持有期收益可以表示为：

$$\log S_{t+h}-\log S_t\approx\mu h+\sigma(W_{t+h}-W_t)$$

根据增量性质，$W_{t+h}-W_t\sim N(0,h)$，方差正比于 $h$。根据缩放性质，不同时间尺度的收益（比如日、周、月）在**形状上是相似的**：只是标准差按 $\sqrt h$ 的比例伸缩。

因此，如果现实市场近似遵循这样的过程，我们就会看到：日收益、周收益、月收益的分布轮廓大体相似。区别主要体现在波动大小上：当我们把观察区间从单日扩展到 $k$ 日时，$k$ 日收益的方差大致与 $k$ 成正比，标准差大致按 $\sqrt{k}$ 的比例放大。这就是 “缩放性质” 与 “不同时间尺度收益分布之间相似性” 之间的联系所在。



设某资产的对数价格为 $X_t = \ln P_t$。我们做一个理想化假设：

$$X_t = X_0 + \mu t + \sigma W_t$$

其中：

* $\mu$：单位时间的期望对数收益率（漂移项）；
* $\sigma$：单位时间的波动率（对数价格的标准差参数）；
* $W_t$：标准布朗运动。

对任意 $0 \le s < t$，记 $h = t - s$，有

$$X_t - X_s = \mu h + \sigma \bigl(W_t - W_s\bigr)$$

因为布朗运动的增量满足 $W_t - W_s \sim N\bigl(0,\,h \bigr)$，所以对数收益 $X_t - X_s = \ln P_t - \ln P_s$ 服从正态分布：

$$X_t - X_s \sim N\bigl(\mu h,\,\sigma^2 h\bigr)$$

亦可表示为：

$$X_{t+h} - X_t \sim N\bigl(\mu h,\,\sigma^2 h\bigr)$$

这就是很多金融模型中常见的设定：**在该模型下，对数收益服从正态分布，其方差与时间跨度成正比**。

## 4. 金融里的布朗运动：从收益到几何布朗运动

### 4.1 连续时间随机游走：收益服从布朗运动

设某资产的对数价格为 $X_t = \ln S_t$，我们可以假设：

$$X_t = X_0 + \mu t + \sigma W_t$$

* $\mu$：长期平均增长率（漂移项）；
* $\sigma$：波动率；
* $W_t$：标准布朗运动。

则对数收益在时间间隔 $\Delta t$ 内为：

$$X_{t+\Delta t} - X_t = \mu \Delta t + \sigma (W_{t+\Delta t} - W_t)$$

由于 $W_{t+\Delta t} - W_t \sim N(0,\Delta t)$，所以对数收益是正态分布：

$$X_{t+\Delta t} - X_t \sim N(\mu \Delta t,\sigma^2 \Delta t)$$

这就是很多金融模型中常见的“收益正态分布 + 方差与时间成正比”的设定。

### 4.2 几何布朗运动：Black–Scholes 模型里的主角

如果直接对价格建模，最常见的是几何布朗运动：

$$dS_t = \mu S_t dt + \sigma S_t dW_t$$

其解为：

$$S_t = S_0 \exp\Big((\mu - \tfrac{1}{2}\sigma^2)t + \sigma W_t\Big)$$

因此：

* $\ln S_t$ 服从正态分布；
* $S_t$ 服从对数正态分布；
* 价格始终为正，这与现实较为贴近。

这正是 Black–Scholes 期权定价模型中的核心假设之一。

## 5. 一个简单的概率例子：一年后收益为正的概率

假设某资产的对数价格满足：

$$X_t = X_0 + \mu t + \sigma W_t$$

比如：

* 年化漂移率 $\mu = 0.08$；
* 年化波动率 $\sigma = 0.2$；
* 我们关心一年后的对数收益 $R = X_1 - X_0$。

根据前面的结论，有：

$$R \sim N(\mu,\sigma^2) = N(0.08,0.04)$$

我们想问：“一年后的对数收益大于0（即价格上涨）的概率是多少？”

也就是：

$$P(R > 0) = P\Big(\frac{R - 0.08}{0.2} > \frac{0 - 0.08}{0.2}\Big) = P(Z > -0.4)$$

其中 $Z \sim N(0,1)$，所以

$$P(Z > -0.4) = P(Z < 0.4) \approx 0.6554$$

也就是说，在这个简单模型下，一年后价格上涨的概率大约为0.66。

这个例子告诉我们：

* 布朗运动让我们能把“随机收益”精确地写成正态分布；
* 再用标准正态分布表或软件，就可以算各种概率。

## 6. 用 Stata 模拟一条“近似布朗运动”的轨迹

在计算机里无法直接生成连续时间的布朗运动，但可以用“细时间步的随机游走”近似。
下面给一段简单的 Stata 代码，模拟标准布朗运动 $W_t$ 的一条路径，并画出曲线。你可以直接用来上课示范。

### 6.1 Stata 模拟代码（离散步长近似布朗运动）

```stata
*======================================================
* 用 Stata 模拟一条标准布朗运动轨迹
*======================================================

clear all
set more off

*-----------------------------
* 1. 设定模拟参数
*-----------------------------
local T     = 1           // 总时间长度，取 1 年
local Nstep = 1000        // 把 [0,T] 划分为 1000 个小步
local Nobs  = `Nstep' + 1 // 需要的观测点数

set obs `Nobs'

gen t = (_n - 1)*(`T'/`Nstep')   // 时间点 t_0,...,t_N

*-----------------------------
* 2. 生成独立增量 ~ N(0, dt)
*-----------------------------
local dt = `T'/`Nstep'

set seed 123456                 // 固定随机种子，结果可重复
gen dW = rnormal(0, sqrt(`dt')) // 每一步的增量 N(0, dt)

* 第一个时间点 t=0 的增量设为 0，方便累计求和
replace dW = 0 in 1

*-----------------------------
* 3. 构造布朗运动轨迹：W_t = 累积和(dW)
*-----------------------------
gen W = .
replace W = 0 in 1              // W_0 = 0

forvalues i = 2/`Nobs' {
    replace W = W[_n-1] + dW in `i'
}

*-----------------------------
* 4. 绘制轨迹图
*-----------------------------
twoway ///
    (line W t, lwidth(medthick)) ///
    , ///
    title("模拟标准布朗运动軌迹") ///
    xtitle("时间 t") ///
    ytitle("W_t") ///
    legend(off) ///
    yline(0, lpattern(dash))
```

你可以在课堂上做几件事：

* 多次运行上面的代码，比较不同随机种子下轨迹的形状；
* 把 `Nstep` 从1000换成100、10000，看看时间步长越细，轨迹越“平滑但粗糙”；
* 把 `W` 换成 `X = μ*t + σ*W`，再画出对数价格或价格的轨迹，连接到金融应用上。

## 7. 小结：如何把布朗运动“记在心里”

可以用一句话来概括：

> 布朗运动是一条连续时间的随机游走曲线：
> 任意小时间步的变化服从正态分布、互相独立，
> 累积起来就是一条“连续但处处抖动”的轨迹。

在金融学中，你可以这样理解它的角色：

* 在“收益层面”，布朗运动是“连续时间的高斯噪声”；
* 在“价格层面”，几何布朗运动是“对数正态价格”的自然模型；
* 在期权定价、利率期限结构模型、风险管理中，布朗运动提供了一个既简单又可计算的随机扰动基础。
