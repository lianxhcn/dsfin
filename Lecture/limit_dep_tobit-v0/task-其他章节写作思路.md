

前面已经介绍了 Tobit 模型，后面要介绍 Two-part model，hurdle model 和 Heckman Sample selection model，Treatment effect model (模型右侧的处理变量 $D$ 存在样本选择或自选择)。这些模型都是为了处理因变量存在截断或者选择性偏差的问题而提出的。

这些模型的介绍无需讲解估计过程，要以应用为导向，重点说明这些模型的适用场景、模型设定和结果解释。可以通过一些实际案例来说明这些模型的应用，比如在金融和公司金融领域，

- Tobit 模型可以用来分析投资金额的决定因素，
- Two-part model 可以用来分析公司在数字化转型中的投资决策，
- hurdle model 可以用来分析企业创新活动的决策过程，
- Heckman Sample selection model 可以用来分析企业的捐赠行为或数字化创新投入 (有一定比例的企业的 $y$ 是缺失值)
- ppmlhdfe 模型可以用来分析企业的贸易流量或者投资流量，处理零值和高维固定效应的问题。


## 应用文献

每个模型提供 1-2 篇应用文献的引文信息，简要介绍文献的研究问题、数据来源、模型设定和主要发现。

- 目的：通过具体的应用案例来说明这些模型的适用场景和结果解释，帮助学生更好地理解这些模型的实际应用价值。
- 文献选择：可以选择一些经典的或者近期的文献来说明这些模型的应用。比如 JF，JFE，QJE，JCF，JBF 等顶级期刊上的文章，或者一些领域内具有代表性的研究。选择文献时可以考虑以下几个方面：
  - 研究问题的相关性：选择与企业金融、投资决策、创新活动等相关的研究问题，能够更好地与课程内容结合。
  - 研究问题的难度：研究背景不必太复杂，只要能清晰地说明研究问题和模型应用即可，避免过于技术性的细节。
  - 重点：呈现模型的设定，重点说明作者为何要使用该模型。
    - 对于 two-part model 和 hurdle model，重点说明为什么要区分零值和正值的决策过程。
    - 对于 Heckman Sample selection model，重点说明样本选择问题的来源和解决方法。
    - 对于 Treatment effect model，重点说明处理变量的选择和潜在的自选择问题。
