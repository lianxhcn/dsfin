## 投资组合优化实战：约束条件、换手成本与绩效归因

### 任务背景

T1 构造了等权重（1/N）投资组合，并初步计算了组合的 Beta 和夏普比率。但在实际资产管理中，组合优化要复杂得多——需要加入权重约束、行业中性、换手成本等现实条件，优化结果才有可落地性。

本题在 T1 的基础上，系统引入**约束优化**框架，将"教科书上的有效前沿"升级为更接近实战的投资组合构建流程，并通过**绩效归因**分解收益来源。

**2. 五类组合的构造与比较**

依次构造以下 5 类组合，并在统一的样本外区间（最近 1 年）上计算绩效：

| 编号 | 组合名称 | 约束条件 |
|------|---------|---------|
| P1 | 等权重（基准） | $w_i = 1/N$ |
| P2 | 最小方差 | $\min w^T \Sigma w$，$\sum w_i = 1$，$w_i \geq 0$ |
| P3 | 最大夏普比率 | $\max \frac{w^T \mu - r_f}{\sqrt{w^T \Sigma w}}$，$\sum w_i = 1$，$w_i \geq 0$ |
| P4 | 约束最大夏普（实战版） | 在 P3 基础上加入：权重上限、行业中性、换手限制 |
| P5 | 风险平价 | 每只股票的风险贡献相等 |

**3. P2/P3：无约束优化（教科书版）**

使用 `scipy.optimize.minimize` 实现：

```python
from scipy.optimize import minimize
import numpy as np

def portfolio_variance(w, cov):
    return w @ cov @ w

def neg_sharpe(w, mu, cov, rf=0.025/12):
    ret = w @ mu
    vol = np.sqrt(w @ cov @ w)
    return -(ret - rf) / vol

# 约束：权重之和为 1，权重非负
constraints = [{"type": "eq", "fun": lambda w: np.sum(w) - 1}]
bounds = [(0, 1)] * n_assets

result = minimize(neg_sharpe, x0=np.ones(n_assets)/n_assets,
                  args=(mu, cov),
                  method="SLSQP",
                  bounds=bounds,
                  constraints=constraints)
```

- 绘制有效前沿曲线（蒙特卡洛模拟 + 优化前沿）
- 标注 P2（最小方差）、P3（最大夏普）、P1（等权重）在有效前沿图上的位置

**4. P4：加入实战约束**

在 P3 基础上，依次加入以下约束，观察每个约束对权重和绩效的影响：

**约束 A：单股权重上限**

$$w_i \leq w_{\max}, \quad \forall i \quad (\text{建议} w_{\max} = 0.1 \text{ 或 } 0.15)$$

```python
bounds = [(0, 0.10)] * n_assets   # 每只股票最多持有 10%
```

**约束 B：行业中性（相对基准）**

$$\left| \sum_{i \in \text{行业}_k} w_i - b_k \right| \leq \delta_k, \quad \forall k$$

其中 $b_k$ 为沪深 300 中该行业的权重，$\delta_k$ 为允许的偏离度（建议 5%）：

```python
# 对每个行业添加约束
for industry, idx in industry_groups.items():
    benchmark_weight = benchmark_weights[industry]
    constraints.append({
        "type": "ineq",
        "fun": lambda w, idx=idx, bw=benchmark_weight:
               0.05 - abs(w[idx].sum() - bw)
    })
```

**约束 C：换手率限制**

$$\sum_i |w_i^{(t)} - w_i^{(t-1)}| \leq \text{TO}_{\max} \quad (\text{建议} \text{TO}_{\max} = 0.3)$$

```python
prev_weights = w_prev  # 上期末权重（再平衡前）
constraints.append({
    "type": "ineq",
    "fun": lambda w: 0.30 - np.sum(np.abs(w - prev_weights))
})
```

- 分步加入约束，记录每步加入后最大夏普比率的变化
- 绘制加约束前后的权重对比图（条形图），直观展示约束如何改变持仓结构

**5. P5：风险平价组合**

风险平价要求每只股票对组合总风险的**边际贡献**相等：

$$RC_i = w_i \cdot \frac{(\Sigma w)_i}{w^T \Sigma w} = \frac{1}{N}, \quad \forall i$$

```python
def risk_parity_objective(w, cov):
    portfolio_vol = np.sqrt(w @ cov @ w)
    marginal_rc = cov @ w / portfolio_vol
    rc = w * marginal_rc          # 各资产风险贡献
    rc_target = portfolio_vol / len(w)
    return np.sum((rc - rc_target)**2)   # 最小化风险贡献的差异
```

- 求解风险平价权重，与等权重（P1）和最小方差（P2）对比
- 讨论：风险平价与等权重的区别在哪里？什么情况下两者相近？

**7. 绩效归因（Brinson 模型）**

对 P4（约束版最大夏普）做 Brinson 行业归因，分解超额收益来源：

$$\text{超额收益} = \underbrace{\sum_k (w_k - b_k) \cdot b_k^r}_{\text{行业配置效应}} + \underbrace{\sum_k b_k \cdot (w_k^r - b_k^r)}_{\text{个股选择效应}} + \text{交互效应}$$

其中 $w_k$、$b_k$ 为组合和基准在行业 $k$ 的权重，$w_k^r$、$b_k^r$ 为行业内收益率。

- 计算每个行业的配置效应和选择效应

- 讨论：超额收益主要来自行业配置还是个股选择？
