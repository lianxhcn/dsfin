- 面板数据模型：固定效应、辛普森悖论、FWL 定理、混淆变量
- 高维固定效应模型：reghdfe
  - 应用：$y_{it} = \alpha_i + \lambda_t + \beta x_{it} + \epsilon_{it}$，其中 $\alpha_i$ 是个体固定效应，$\lambda_t$ 是时间固定效应，$\beta$ 是我们关心的系数。
  - 国际贸易中的引力模型：$Trade_{ijst} = \alpha_i + \alpha_j + \alpha_s + \alpha_t + \beta Distance_{ij} + \epsilon_{ij}$，其中 $\alpha_i$ 是出口国固定效应，$\lambda_j$ 是进口国固定效应，$alpha_s$ 是行业固定效应，$\alpha_t$ 是年度效应， $\beta$ 是距离的系数。
- 交互固定效应，Bai (2009) 模型, `regife` 命令
  - 应用：$y_{it} = \alpha_i + \beta x_{it} + \theta EPU_t + \epsilon_{it}$ 这类混频数据的回归，重点关注的是 $\theta$。为了防止遗漏其他时间变化的因素，我们可以加入交互固定效应 $\alpha_i \times \lambda_t$ 来控制这些因素，否则的得到的 $\theta$ 就是有偏的。
- 面板门槛模型：Panel threshold model (Hansen, 1999, JoE; Wang, 2015, SJ)
- 面板函数系数模型：`xtplfc` 或者采用 `reghdfe` 并加入高阶调节项

# 习题设计想法

## 上市公司资本结构影响因素及结构变化

### 研究问题

验证资本结构理论中的权衡理论和优序融资理论，分析上市公司资本结构的影响因素，并检验其结构是否发生了变化。

- 二者的核心差别在于 Lev 和 NPR (Net Profit Rate = NetIncome/TotalAssets) 之间的关系。
  - 权衡理论认为 Lev 和 NPR 正相关，因为利润越高，企业越有能力承担更多的债务；银行等债权人也更愿意借钱给利润高的企业。
  - 优序融资理论认为 Lev 和 NPR 负相关，因为利润越高，企业越倾向于使用内部资金进行投资，而不是通过债务融资。
  - 权衡理论强调的是企业融资可得性，即净利润越高，企业越有能力承担更多的债务；优序融资理论强调的是企业融资偏好 (或融资需求)，即净利润越高，企业越倾向于使用内部资金进行投资。

- 调节效应：
  - SOE (国有企业和民营企业)：国有企业可能更倾向于使用债务融资，因为它们通常具有更好的融资可得性；民营企业可能更倾向于使用内部资金进行投资，因为它们通常面临更大的融资约束。
    - 模型设定：通过引入 SOE 与 NPR 的交互项来检验所有权性质对 NPR-Lev 关系的调节作用。
  - 信息不对称：优序融资理论的理论基础是信息不对称，因此可以通过引入信息不对称的调节变量来检验信息不对称对 NPR-Lev 关系的调节作用。例如，可以使用企业规模 (Size) 来衡量信息不对称程度，规模较大的企业通常信息更透明，信息不对称程度较低；规模较小的企业通常信息更不透明，信息不对称程度较高。
    - 模型设定：通过引入 Size 与 NPR 的交互项来检验信息不对称程度对 NPR-Lev 关系的调节作用；进一步，可以将 Size 设定为 NPR 的函数来检验信息不对称程度对 NPR-Lev 关系的调节作用，例如 $\beta(Size_{it})$。

### 变量设计

- 核心变量定义：
  - Leverage = TotalDebt/TotalAssets，衡量企业的资本结构。
  - NPR (Net Profit Rate) = NetIncome/TotalAssets，衡量企业的盈利能力。

- 控制变量
  - Size = log(TotalAssets)，企业规模，通常认为规模较大的企业更容易获得债务融资。
  - Tangibility = TangibleAssets/TotalAssets，资产结构，通常认为有更多有形资产的企业更容易获得债务融资。
  - Growth = (TotalAssets_t - TotalAssets_{t-1})/TotalAssets_{t-1}，企业成长性，通常认为成长性较高的企业更倾向于使用内部资金进行投资。
  - Industry：控制行业特征的混淆影响。
    - 行业要根据证监会行业分类标准进行划分，常见的有 1 位数、2 位数、3 位数的行业分类，具体选择哪一种取决于数据的可用性和研究的需要。
    - 本作业中要求：
      - 除制造业以外的其他行业按照 1 位数的行业分类进行划分，制造业按照 2 位数的行业分类进行划分。
      - 若有些行业的样本量过小（例如某些 2 位数的制造业行业），可以考虑将这些行业合并到一个“其他制造业”类别中，以保证每个行业类别都有足够的样本量进行分析。
  - Year：控制时间特征的混淆影响。
  - 所有权性质 (SOE)：国有企业-1、民营企业-0，控制所有权性质对资本结构的影响。

-   **非债务税盾（Non-Debt Tax Shield, NDTS）**
   -   计算公式：折旧与摊销 / 总资产
   -   理论依据：折旧抵税可替代债务税盾，与杠杆负相关，是权衡理论关键变量。
-   **流动性（Liquidity）**
  -   计算公式：流动资产 / 流动负债
  -   理论依据：流动性越强，企业短期偿债能力越强，同时内源资金更充足，影响杠杆选择。
-   **经营风险 / 盈利波动性（Risk / Volatility）**
  -   计算公式：ROA 近 3 年标准差（或息税前利润波动率）
  -   理论依据：风险越高，财务困境成本越大，权衡理论下企业会降低负债水平。
-   **股权集中度（Ownership Concentration）**
  -   计算公式：第一大股东持股比例
  -   理论依据：股权集中程度影响代理成本与融资决策，是中国上市公司重要特征。
-   **实际税率（Effective Tax Rate, ETR）**
  -   计算公式：所得税费用 / 利润总额
  -   理论依据：税率越高，债务税盾价值越大，权衡理论下杠杆水平越高。

### 数据来源和处理

- CSMAR：在中山大学 IP 地址范围内可以访问，提供了中国上市公司的财务数据和市场数据。
  - 自行登录 CSMAR 网站，下载相关数据。
  - Notes：上述指标可能分布在不同的数据库中，例如财务数据、市场数据等，需要分别下载并进行合并处理。
- 数据处理：要删除曾经被 ST/PT 处理过的公司，因为这些公司的财务数据可能存在异常，可能会对回归结果产生较大的影响。
  - 具体来说，可以通过 CSMAR 数据库中的 ST/PT 标志来识别这些公司，并在数据清洗阶段将它们从样本中剔除。
- 数据处理：删除金融和保险行业的公司，因为这些行业的资本结构特征与其他行业有显著不同，可能会对回归结果产生较大的影响。
  - 具体来说，可以通过 CSMAR 数据库中的行业分类代码来识别金融和保险行业的公司，并在数据清洗阶段将它们从样本中剔除。
- 数据处理：需要对数据进行清洗和处理，例如去除缺失值、处理异常值、计算相关指标等。
  - 例如，关键财务指标通常需要先做描述性统计量分析，检查数据的分布情况和异常值，然后根据需要进行 Winsorize 或者其他处理方法来处理异常值。也可以绘制处理前后的箱线图来直观展示异常值的处理效果。
  - 已经进行了对数转换的变量，如 Size (ln(TotalAssets))，通常是不需要进行 Winsorize 的，因为对数转换已经在一定程度上缓解了极端值的影响。
  - SOE，行业分类这些类别变量通常也不需要进行 Winsorize，因为它们本身就是分类变量，不存在极端值的问题。




### 模型设计

- 基础模型：Leverage_{it} = \alpha_i + \lambda_t + \beta NPR_{it} + \gamma X_{it} + \epsilon_{it}
  - 其中，\alpha_i 是个体固定效应，\lambda_t 是时间固定效应，X_{it} 是控制变量。
  - 重点关注：\beta 的符号和显著性，以验证权衡理论和优序融资理论的预测。
- 分组回归：
  - 根据所有权性质进行分组回归，检验不同组别之间的关系是否存在差异。
  - 根据行业进行分组回归，检验不同行业之间的关系是否存在差异。
    - 建议不要把每个行业都做一遍，而是选择几个具有代表性的行业划分方法进行分析，例如：
      - M1：区分制造业和非制造业，或者区分高科技行业和传统行业。
      - M2：区分市场化程度较低 (如能源、电力；交通运输等) 和市场化程度较高的行业
- 调节效应分析：
  - 通过引入交互项来检验调节效应，例如：
    - Leverage_{it} = \alpha_i + \lambda_t + \beta NPR_{it} + \delta (NPR_{it} \times SOE_i) + \gamma X_{it} + \epsilon_{it}
    - 其中，\delta 的符号和显著性可以检验所有权性质对 NPR-Leverage 关系的调节作用。
- 时变系数模型
  - 允许 $\beta$ 随年度变化，并绘制更年度的 $\beta_t$ 的时间趋势图来直观展示资本结构影响因素的结构变化，例如：
    - Leverage_{it} = \alpha_i + \lambda_t + \beta_t NPR_{it} + \gamma X_{it} + \epsilon_{it}
    - 通过绘制 $\beta_t$ 的时间趋势图来直观展示 NPR 对 Leverage 影响的时序特征。
  - 进一步允许所有控制变量系数可以随年度变化，并绘制此种设定下 $\beta_t$ 的时序图：
    - Leverage_{it} = \alpha_i + \lambda_t + \beta_t NPR_{it} + \sum_k \gamma_{k,t} X_{k,it} + \epsilon_{it}
    - 通过绘制 $\beta_t$ 的时间趋势图来直观展示 NPR 对 Leverage 影响的时序特征，同时控制了其他变量系数随时间变化的影响。
- 函数系数模型
  - 参考 [面板数据-变系数模型](https://lianxhcn.github.io/dsfin/Lecture/panel_xtplfc/04_xtplfc%E7%9A%84%E7%9B%B4%E8%A7%82%E8%A7%A3%E8%AF%BB%E5%8F%8Axthreg%E5%AE%9E%E7%8E%B0.html) 中的设定，将 $\beta$ 设定为 Size (用来衡量信息不对称) 的函数，例如：
  - $Leverage_{it} = \alpha_i + \lambda_t + \beta(Size_{it}) NPR_{it} + \gamma X_{it} + \epsilon_{it}$
  - $\beta(Size_{it})$ 可以设定为一个线性函数，例如 $\beta(Size_{it}) = \beta_0 + \beta_1 Size_{it}$，也可以设定为一个非线性函数，例如 $\beta(Size_{it}) = \beta_0 + \beta_1 Size_{it} + \beta_2 Size_{it}^2 + \beta_3 Size_{it}^3$。
  - 使用 `margins` 和 `marginsplot` 命令来绘制 $\beta(Size_{it})$ 的函数图像，直观展示信息不对称程度对 NPR-Leverage 关系的调节作用。
