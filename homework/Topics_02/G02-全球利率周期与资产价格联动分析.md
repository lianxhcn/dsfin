### 任务背景

利率是宏观经济中最重要的价格信号之一。当美联储开启加息或降息周期，影响往往不局限于美国境内——其他经济体的汇率、通胀和资产价格都会随之波动，只是幅度和方向因国而异。

本题以**全球利率周期**为主线，整合四个数据源的宏观与市场数据，围绕三个分析维度展开：

- **维度 1（利率 → 通胀）**：加息周期中各经济体通胀的响应路径是否一致？
- **维度 2（利率 → 汇率）**：加息周期中各经济体货币对美元的升贬值规律
- **维度 3（利率 → 股市）**：加息周期对主要股指收益率及波动率的影响

数据整合本身就是这道题的核心挑战：四个来源的数据格式不同、频率不同、国家命名方式不同、覆盖时段不同。在分析开始之前，你必须先建立一套干净、统一、可查询的数据库。

**研究对象**：以下 6 个经济体，时间范围 **1990 年至今**（部分指标可能更短）：

| 经济体 | ISO3 代码 | 选取理由 |
|--------|---------|---------|
| 美国 | USA | 全球利率锚，美联储政策的直接来源 |
| 中国 | CHN | 最大新兴市场，货币政策相对独立 |
| 德国/欧元区 | DEU | 欧洲代表，欧央行政策 |
| 日本 | JPN | 长期超宽松、负利率的典型案例 |
| 英国 | GBR | 独立货币政策，脱欧背景 |
| 巴西 | BRA | 高利率新兴市场，对美联储政策高度敏感 |

**路径 B：直接调用世界银行 REST API**

```python
import requests, pandas as pd

def get_wb(indicator, countries, start=1990, end=2024):
    iso = ";".join(countries)
    url = (f"https://api.worldbank.org/v2/country/{iso}"
           f"/indicator/{indicator}"
           f"?date={start}:{end}&format=json&per_page=1000")
    r = requests.get(url, timeout=30)
    data = r.json()[1]
    return pd.DataFrame([{
        "ISO3":  d["countryiso3code"],
        "year":  int(d["date"]),
        "value": d["value"]
    } for d in data if d["value"] is not None])
```
