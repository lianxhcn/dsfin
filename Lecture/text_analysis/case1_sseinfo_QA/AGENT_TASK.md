## 5. 回复质量分析：董秘答得怎么样（§5）

> 本节是本案例的核心创新部分，引入**跨文档相似度矩阵**来检测系统性回复问题。
> 所有分析仅针对**有回复**的记录（`replied == True`）。

### 5.1 问答相关性（答了吗？）

**方法**：对每个 Q-A 对，分别计算提问和回复的 TF-IDF 向量，
再计算两者的余弦相似度，存为 `qa_similarity` 列。

步骤：

1. 对 `question` 和 `answer` 分别分词，拼成空格分隔的字符串
2. 合并两列，构建统一的 `TfidfVectorizer`
3. 对每一对 (Q_i, A_i)，计算余弦相似度
4. 生成 `5a_qa_similarity_dist.png`：相似度分布直方图，标注均值和中位数

**深挖低相似度记录**：

- 找出 `qa_similarity` 最低的 10 条，打印原始提问和回复
- 在 Markdown 中写 2–3 句分析：这些"低相关性回复"有什么共同特征？
  是真的答非所问，还是因为回复极短（如"感谢关注"）导致的低相似度？

### 5.2 回复的复制粘贴检测（有没有偷懒？）

**方法**：计算所有**回复**之间的两两余弦相似度，构建相似度矩阵。

步骤：

1. 只取有回复记录的 `answer` 列，构建 TF-IDF 矩阵（记录之间）
2. 计算 n×n 的余弦相似度矩阵（排除对角线）
3. 生成 `5b_answer_similarity_heatmap.png`：相似度矩阵热图
   （如记录数过多，随机抽取 200 条）
4. 统计：相似度 > 0.8 的回复对有多少组？占总回复对的比例？

**高相似度回复对的内容分析**：

- 找出相似度最高的 5 组回复对，打印对应的提问和回复
- 判断：这些提问的主题（`topic_llm`）是否相同？
  还是跨主题的"万能回复"？
- 在 Markdown 中写你的发现和判断

### 5.3 跨主题回复同质性（特定话题被敷衍了？）

**方法**：按 `topic_llm` 分组，分别计算组内回复的平均相似度，
并与随机配对的基准值比较。

步骤：

1. 对每个主题分组，计算该主题所有回复对的平均余弦相似度
2. 对照组：在全体回复中随机抽取等量配对，计算平均相似度
3. 生成 `5c_topic_reply_similarity.png`：各主题回复同质性柱状图
   （每个柱子代表该主题组内平均相似度，叠加对照组的参考线）
4. 找出回复同质性最高的 2–3 个主题，阅读原文，
   判断是否存在系统性的"模板回复"倾向

**综合评分**（可选，加分项）：

为每条回复构造一个简单的"回复质量得分"：

```python
quality_score = (
    0.5 * qa_similarity          # 与提问的相关性（越高越好）
  - 0.5 * max_answer_similarity  # 与其他回复的相似性（越低越好）
)
```

- 统计得分的分布，找出得分最高和最低的各 5 条，阅读原文验证
- 生成 `5d_quality_score_dist.png`：质量得分分布图
