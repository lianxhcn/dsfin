## 2.1 总体、样本与经验分布

### 2.1.1 总体（Population）

**案例锚点**：沪深300指数日收益率——"真实的总体"是什么？

- 总体是研究对象的全集，由概率分布 $Y \sim F(\cdot;\theta)$ 刻画
- 金融中总体往往是**概念性**的：所有可能发生的交易日收益率
- 参数 $\theta$ 是我们想知道但永远无法精确知道的量

### 2.1.2 样本（Sample）

- IID假设：独立同分布——金融时间序列为何天然违反这一假设？
  - 波动率聚集（ARCH效应）作为直观反例
- 样本量 $n$ 的代价：信息 vs. 平稳性的取舍（用5年 vs. 20年数据）
- **统计量（Statistic）**：$T = g(y_1, \ldots, y_n)$，是对样本信息的压缩

### 2.1.3 经验分布（Empirical Distribution）

$$\hat{F}_n(x) = \frac{1}{n}\sum_{i=1}^n \mathbf{1}(y_i \leq x)$$

- Glivenko-Cantelli 定理（直觉）：随着 $n \to \infty$，$\hat{F}_n \to F$
- ECDF 是后续核密度估计、Bootstrap的基础

## 2.2 分布的三种角色

本章核心框架，也是频率派与贝叶斯派分野的根源：

| 角色           | 记号                        | 直觉                     | 后续章节            |
| -------------- | --------------------------- | ------------------------ | ------------------- |
| **数据分布**   | $y_i \sim F(\cdot\|\theta)$ | 数据从哪个"瓮"里抽出来的 | 所有回归模型        |
| **参数分布**   | $\theta \sim \pi(\cdot)$    | 参数本身的不确定性       | Bayes推断、多层模型 |
| **误差项分布** | $\epsilon_i \sim G(\cdot)$  | 模型"解释不了"的部分     | OLS、GLM            |

### 2.2.1 数据分布

- 以上市公司违约为例：$y_i \in \{0,1\}$，自然对应 Bernoulli 分布
- 以日交易笔数为例：$y_i \in \{0,1,2,\ldots\}$，自然对应 Poisson 分布
- **选错分布的代价**：用正态模型估计违约率会发生什么？

### 2.2.2 参数分布

- 频率派：$\theta$ 是固定的未知常数，没有分布
- 贝叶斯派：$\theta$ 本身是随机变量，有先验分布 $\pi(\theta)$
- **金融直觉**：A股各行业股票的 $\beta$ 系数——把它们看作来自某个分布，还是各自独立的常数？

### 2.2.3 误差项分布

- $\epsilon_i = y_i - \mathbf{x}_i^\top\beta$：正态假设从何而来（CLT的辩护）
- 正态误差假设的局限：金融收益率的厚尾意味着什么？
- 替代选择：$t$ 分布误差、异方差稳健标准误

## 2.3 分布的数字特征

### 2.3.1 矩（Moments）

- 均值 $\mu$、方差 $\sigma^2$、偏度 $\gamma_1$、（超额）峰度 $\gamma_2$
- 金融数据的"典型签名"：**左偏（负偏度）+ 厚尾（峰度 > 3）**

$$\gamma_1 = \frac{E[(Y-\mu)^3]}{\sigma^3}, \quad \gamma_2 = \frac{E[(Y-\mu)^4]}{\sigma^4} - 3$$

- Jarque-Bera 正态性检验：基于偏度和峰度

### 2.3.2 分位数与风险度量

- $p$ 分位数 $q_p$：$F(q_p) = p$
- **VaR**（风险价值）：$\text{VaR}_\alpha = -q_\alpha$，是监管与风控的核心工具
- **CVaR / Expected Shortfall**：$E[Y | Y < q_\alpha]$，比VaR更能刻画尾部风险
- 正态假设下的VaR vs. 历史模拟法的VaR——哪个更保守？

### 2.3.3 矩母函数简介（选读）

- $M_Y(t) = E[e^{tY}]$：统一刻画所有矩的工具
- 对推导CLT有用，对贝叶斯共轭分布有用

**📓 Notebook 示例 2.3**：用2005-2023年沪深300数据，计算滚动偏度和峰度，展示金融危机期间尾部特征如何显著变化；同时对比正态假设下的VaR与历史模拟VaR，量化"分布误设"的代价。

## 2.4 多变量分布基础

### 2.4.1 联合、边际与条件分布

$$f_{X,Y}(x,y) = f_{Y|X}(y|x) \cdot f_X(x)$$

- **回归的分布语言**：OLS 建模的是 $E[y|\mathbf{x}]$，即条件期望
- 独立 vs. 不相关：正态分布是特例（不相关即独立）

### 2.4.2 协方差、相关系数与依赖结构

- Pearson相关 $\rho$：度量线性依赖
- Spearman秩相关：对非线性依赖更稳健
- **金融危机中的相关结构**：平时低相关的资产，危机时相关性骤升——Pearson相关抓不住这个现象

### 2.4.3 多元正态分布

$$\mathbf{Y} \sim \mathcal{N}(\boldsymbol{\mu}, \boldsymbol{\Sigma})$$

- 良好的封闭性：边际正态，条件正态
- 协方差矩阵 $\boldsymbol{\Sigma}$ 的经济含义：资产组合理论的基础
- 局限：无法刻画非对称的尾部依赖（Copula 模型的动机，选读）

## 2.5 大数定律与中心极限定理

### 2.5.1 大数定律（LLN）

$$\bar{Y}_n \xrightarrow{p} \mu \quad \text{as } n \to \infty$$

- 直觉：样本均值以概率收敛于总体期望
- 意义：估计量的**一致性**（Consistency）——推断合法性的根基
- 金融应用：用历史均值估计预期收益率需要多少年数据？

### 2.5.2 中心极限定理（CLT）

$$\sqrt{n}\left(\bar{Y}_n - \mu\right) \xrightarrow{d} \mathcal{N}(0, \sigma^2)$$

- 意义：无论总体分布如何，样本均值渐近正态——这是 $t$ 检验、置信区间、标准误的理论基础
- **厚尾分布的挑战**：$t$ 分布族（$\nu < 2$ 时方差不存在）收敛极慢
- 有限样本下的近似质量——$n = 30$ 的"经验法则"在金融数据中是否成立？

## 2.6 金融建模中的常用分布

**核心导图**（每个分布：定义 → 参数 → 矩的特征 → 金融语境 → 对应模型）：

### 连续分布

| 分布                             | 支撑                | 金融语境           | 对应模型  |
| -------------------------------- | ------------------- | ------------------ | --------- |
| 正态 $\mathcal{N}(\mu,\sigma^2)$ | $(-\infty,+\infty)$ | 收益率近似         | OLS       |
| $t(\nu)$                         | $(-\infty,+\infty)$ | 厚尾收益率         | 稳健回归  |
| 对数正态                         | $(0,+\infty)$       | 股价、市值         | —         |
| Gamma / 指数                     | $(0,+\infty)$       | 持仓时长、事件间隔 | 生存分析  |
| Beta                             | $(0,1)$             | 比例、评分         | Beta回归  |
| 截断/审查正态                    | 受限连续            | 受限因变量         | **Tobit** |

### 离散分布

| 分布             | 支撑               | 金融语境             | 对应模型           |
| ---------------- | ------------------ | -------------------- | ------------------ |
| Bernoulli / 二项 | $\{0,1\}$          | 违约、股价涨跌       | **Logit / Probit** |
| Poisson          | $\{0,1,2,\ldots\}$ | 交易笔数、违约事件数 | **Poisson 回归**   |
| 负二项（NB）     | $\{0,1,2,\ldots\}$ | 过离散计数数据       | **NB 回归**        |

## 2.7 从分布到模型：统一视角

### 2.7.1 广义线性模型（GLM）的分布逻辑

GLM 的三要素：

1. **随机成分**：$y_i \sim$ 指数族分布（选定分布 = 选定模型）
2. **系统成分**：线性预测子 $\eta_i = \mathbf{x}_i^\top\beta$
3. **连接函数**：$g(E[y_i]) = \eta_i$

```
选定分布        连接函数          模型
Bernoulli   →  Logit link    →  Logistic 回归
Poisson     →  Log link      →  Poisson 回归
Normal      →  Identity      →  OLS 回归
Gamma       →  Log link      →  Gamma 回归
```

### 2.7.2 两种推断框架的分布观

|                 | 频率派                    | 贝叶斯派                                                  |
| --------------- | ------------------------- | --------------------------------------------------------- |
| $\theta$ 的性质 | 固定未知常数              | 随机变量，有分布                                          |
| 用到的分布      | 仅数据分布 $f(y\|\theta)$ | 数据分布 + 参数先验 $\pi(\theta)$                         |
| 推断工具        | 似然函数 → MLE            | 后验分布 $\pi(\theta\|y) \propto f(y\|\theta)\pi(\theta)$ |
| 不确定性表达    | 置信区间                  | 可信区间（Credible Interval）                             |

### 2.7.3 本章知识地图

```
总体 F(·;θ)
    ↓ 抽样
样本 {y₁,...,yₙ}
    ↓ 压缩
统计量 T（均值、方差、矩）
    ↓ 两条路
    ├── 频率派：L(θ|y) → MLE → 置信区间
    └── 贝叶斯：π(θ|y) ∝ L(θ|y)·π(θ) → 后验分布
```
