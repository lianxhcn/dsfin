### 课程背景

- 课程名称：金融数据分析与建模
- 课程目标：培养学生在金融领域进行数据分析和建模的能力，使他们能够运用数据科学的方法来解决金融领域的实际问题。
- 课程内容：包括数据获取、数据清洗、数据分析、建模方法、模型评估等方面的内容，重点是实用性和应用性，结合金融领域的实际案例来讲解方法的应用。
- 学生未来的职业目标：进入金融行业，尤其是银行、投行、基金、券商等领域，从事金融分析、财务分析、估值、风险管理、量化研究等工作；还有约 1/4 的同学会通过公务员考试进入政府机关。因此，他们更关注课程内容的实用性和应用性，希望能够学到一些能够直接应用到工作中的技能和知识；而不是用来做学术研究的方法。


### 学生背景更新：

- 班上有约 40 名学生，80% 左右都是通过保研方式进入中山大学岭南学院金融硕士项目的，这些学生多数来自 985/211 学校，且大多数学生的本科专业是金融学、经济学、统计学等相关专业。学生的数学基础较好，尤其是在概率论、统计学和线性代数方面有较强的能力。此外，学生们对金融市场和金融工具有一定的了解，具备一定的数据分析能力，能够使用 R、Python 等编程语言进行数据处理和分析。
- 学生的目标：这些学生很少会继续读博士，做科研。他们主要的目标是进入金融行业，尤其是银行、投行、基金、券商等领域，从事金融分析、财务分析、估值、风险管理、量化研究等工作；还有约 1/4 的同学会通过公务员考试进入政府机关。因此，他们更关注课程内容的实用性和应用性，希望能够学到一些能够直接应用到工作中的技能和知识；而不是用来做学术研究的方法。 
- 我想到的一些比较实用的内容：
  - 通过各种 API (比如 akshare, world bank data API, FRED API, CSMAR, Wind 等) 来获取金融数据和经济数据，并进行数据清洗和分析。
  - 掌握数据分析中的常用方法，但对于严谨的因果分析方法他们可能不需要了解的那么细致。反而是常用的机器学习方法应该掌握，并介绍 DDML，神经网络的一些方法。尤其是 Python 中的比较成熟的机器学习库（比如 scikit-learn, TensorFlow, PyTorch 等）在金融领域的应用。换言之，他们最看重的是快速解决问题的内容，而不是过于理论化的内容。如果我能找到一些金融领域的实际案例来讲解这些方法的应用，那就更好了。比如，如何使用机器学习方法来进行股票价格预测、风险管理、客户分类、违约判别、投资组合等。
- 关于案例。我自己寻找大量的优秀案例有比较大的难度，因为如果对某个场景没有深入的了解，很难找到合适的案例来讲解方法的应用。相比之下，学生们可能更熟悉一些金融领域的实际案例，他们也更清楚哪些案例比较有代表性和实用性。因此，我可以让学生们自己寻找一些他们感兴趣的金融领域的实际案例，并在课堂上进行分享和讨论。这样不仅可以让学生们更好地理解方法的应用，还可以激发他们的兴趣和积极性。
  - 所以，我想到的模式是：我上课重点讲解核心原理，然后找一些数据分析方面的项目 (比如来自于 Kaggle，一些偏终于案例和数据分析的 papers)，让学生以小组作业的方式来完成。当然，我也会留一些机会给学生，让他们自己根据在实习期间或课余时间，甚至是别的课程中想到的问题进行数据分析。这样经过 2-3 年的课程积累，我就可以把这些案例报告和分析结果积累成一个比较丰富的案例库，供后续的课程使用。这样不仅可以让学生们更好地理解方法的应用，还可以激发他们的兴趣和积极性，同时也可以为后续的课程提供丰富的教学资源。


经过与学生沟通，我需要对授课内容进行一些调整：

1. 学生已经修过一门「时间序列分析」课程，ARMA，GARCH 等模型都已经学过了。所以 '### Topic 6: 时间序列分析基础' 可以删去
2. Topic 7: 回归分析部分可以保留，但不要采用传统的讲法，而要放在因果推断 (强调条件期望、good controls，backdoor criterion) 的框架下讲解。也就是说，回归分析是为了估计条件期望函数而存在的工具，而不是为了检验线性关系而存在的工具。内容上，包括 CEF 的定义，good controls 的概念，backdoor criterion 的概念，以及如何用回归分析来估计 CEF。重点还包括 FWL 定理，交互项和高维固定效应。重点是介绍基本原理后，通过论文和 JF，JFE 论文上的例子来演示如何应用这些工具来进行因果推断。这样可以让学生更好地理解回归分析在经济学中的实际应用，而不是仅仅停留在理论层面。
   - Event Study 的内容可以作为学生作业的一部分，要求学生使用回归分析来进行事件研究，分析某个事件对某个变量的影响。这样可以让学生更好地理解回归分析在实际中的应用，同时也可以锻炼他们的数据分析能力。
   - 上课时可以简要介绍 Event Study 的基本原理和方法，但不需要过于深入。重点是让学生了解回归分析在事件研究中的应用，而不是让他们掌握所有的细节。
3. Topic 8 的内容有点多，可以拆成两章。
   - Topic 8: 因果推断基础，介绍因果推断的基本概念和方法，包括随机对照试验、自然实验、工具变量法、差分法等。重点是让学生理解这些方法的基本原理和适用场景，以及如何选择合适的方法来进行因果推断。
   - Topic 9: 因果推断进阶，介绍一些更高级的因果推断方法，如合成控制法、回归控制法、合成DID等。重点是让学生了解这些方法的基本原理和适用场景，以及如何应用这些方法来进行因果推断。其实，合成控制法和 SDID 可以用来做 Event Study 的分析。此处，可以用 SCM 和 SDID 再做一次前面讲过的 Event Study 的分析，这样可以让学生更好地理解这些方法在实际中的应用。
